# **Short Summary**
This Plugin adds a chaotic twist to normal pvp arena games. This has SUPER engaging new features such as dimension shift, location swap, power surge and much more to make the normal boring pvp super fun.

# **Features**
1. KIT selection
   (Warrior, Archer and Tank)
2. Changing dimensions
   (shuffle between overworld, nether and end arenas mid fight)
3. Random effects (difficulty scaled)
4. Player position swaps
5. Random chasing mobs (difficulty scaled)
6. Low gravity
7. Increasing difficulty by time
   (higher difficulty = more mob spawn, better mob ai, higher potion amplifier and more)

# **IMPORTANT NOTE**
1. AT LEAST 2 players required or the game will break. 
2. MINECRAFT WORLD FILE IMPORTANT or the game will break. 
Full server file: [Chaos Shift server](https://drive.google.com/file/d/1k1yKTIWL4QCDiK9PgeXqLhHZ5j1Wu3of/view?usp=sharing)

# **Commands**
1. /startchaos: starts the game

2. /stopchaos : force stops the game
